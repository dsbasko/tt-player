# TTS Player

> [English version (README.md)](README.md)

Автономное macOS-приложение в строке меню для озвучивания текста нейронными голосами Microsoft (edge-tts).
Выделите текст в любом приложении, нажмите глобальную горячую клавишу — и услышите его вслух.

## Какую проблему решает

На macOS нет удобного встроенного способа быстро озвучить выделенный текст качественным голосом.
Системный TTS звучит роботизированно, а сторонние решения требуют сложной настройки или подписки.

TTS Player решает это: один хоткей — и любой текст читается вслух нейронным голосом, независимо от того, в каком приложении вы работаете.

## Возможности

- **Глобальная горячая клавиша** — работает в любом приложении (по умолчанию `Ctrl+Cmd+S`)
- **Нейронные голоса Microsoft** — естественное звучание через edge-tts
- **Автоопределение языка** — русский и английский определяются автоматически по доле кириллицы
- **Управление воспроизведением** — play/pause, стоп, перемотка на ±10 сек, скорость (1x–2x)
- **Now Playing** — интеграция с виджетом «Исполняется» и медиа-клавишами
- **Иконка в строке меню** — полное управление без лишних окон
- **Настраиваемый хоткей** — можно переназначить прямо из меню
- **CLI-воспроизведение** — озвучивание буфера обмена из терминала (`tts_player play`)
- **Unix-сокет** — IPC для управления извне

## Как работает

1. Вы нажимаете горячую клавишу
2. Приложение симулирует `Cmd+C` и забирает выделенный текст из буфера обмена
3. Определяется язык текста (русский или английский)
4. edge-tts генерирует аудиофайл через Microsoft Azure Neural TTS
5. AVPlayer воспроизводит аудио с поддержкой управления через медиа-клавиши

## Требования

- macOS 13+
- Python 3.9+
- Разрешение на Универсальный доступ (Accessibility)

## Установка и запуск

```bash
make setup      # Создать Python venv и установить edge-tts
make build      # Скомпилировать бинарник
make run        # Собрать и запустить
make install    # Собрать и установить в /usr/local/bin/
```

Другие команды:

```bash
make play       # Озвучить текст из буфера обмена
make kill       # Остановить запущенный плеер
make uninstall  # Удалить бинарник и директорию данных
make clean      # Удалить локальный бинарник
```

## Архитектура

Однофайловое Swift-приложение (`tts_player.swift`, ~1100 строк), компилируется напрямую через `swiftc` — без Xcode-проекта.
